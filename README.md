# T-SQL-standarts-and-templates

**Проєкт:** Колекція стандартів, шаблонів та навчальних матеріалів для роботи з T-SQL / SQL.

**Мета:** Зібрати структуровані уроки, приклади запитів та корисні скрипти, які допомагають навчатися та стандартизувати стиль написання SQL-запитів у проєктах.

**Структура репозиторію**
- **`course/lessons/`**: основні уроки у форматі Markdown (`.md`).
- **`course/concepts/`**, **`course/exercises/`**: додаткові матеріали та практичні вправи.
- **`assets/images/`**: ілюстрації для уроків (організовано по папках за уроками).
- **`scripts/`**: утиліти для обробки файлів (наприклад, `fix_divs_and_blockquotes_sql.py`, `normalize_bold_sql.py`, `wrap_center_divs.sh`).
- **`archive/`**: архівні або перенесені файли (історія й резервні копії).
- **`backups/`**: додаткові резервні копії (якщо використовуються).

**Як працювати з матеріалами**
- Перегляд уроків: відкрийте файли у `course/lessons/*.md` у редакторі або згенеруйте статичну документацію.
- Редагування: змініть відповідний `.md`, потім виконайте `git add` → `git commit` → `git push`.
- Скрипти для автоматичної обробки: див. `scripts/`.

Приклади команд:

```bash
# Перегляд переліку уроків
ls -la course/lessons

# Запуск скрипту на Python (приклад нормалізації)
python3 scripts/fix_divs_and_blockquotes_sql.py

# Запуск shell-скрипту
bash scripts/wrap_center_divs.sh
```
